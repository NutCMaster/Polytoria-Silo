# Polytoria Silo
### A library used as a resource for polytoria models, currently work in progress


## Why choose polytoria silo?
1) You can contribute (as long as you have an account)!
2) Its free, and your uploaded models come in a day or 6
3) There arent many tutorials that teaches coding, and if you are looking for something, this is a great place to start!


## How do I upload?
1) Open up the polytoria creator.
2) Press file < open
3) And look up your file in the box which says "Search"

### I cant find my file...
Try setting the file format to all, and looking up your file name
in the box that says search.
